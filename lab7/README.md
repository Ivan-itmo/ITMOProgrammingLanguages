# Практическое занятие №7: C# + C + Python

Комплексное взаимодействие между тремя языками:
- **C#** — основное приложение (.NET)
- **C** — высокопроизводительные вычисления через P/Invoke
- **Python** — встраивание интерпретатора CPython через `Python.NET`

---

## Части проекта

### 1. **C → C# (P/Invoke)**
- Сборка нативной библиотеки `native_math.so` / `native_math.dll`
- Вызов `fib_c_recursive` и `fib_c_iterative` из C#
- Сравнение производительности с управляемым кодом

### 2. **Python → C# (Python.NET)**
- Встраивание интерпретатора Python в .NET-процесс
- Импорт модулей (`math`, `builtins`)
- Передача данных между средами
