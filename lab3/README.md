# Практическое занятие №3: ООП в C# — Симулятор боя

Консольное приложение на C#, демонстрирующее основные принципы объектно-ориентированного программирования в экосистеме .NET.

## Описание
Симулятор пошагового боя между двумя героями — **Рыцарем** и **Магом**.  
Каждый герой имеет:
- Имя, здоровье (HP), атаку и защиту
- Обычную атаку
- Уникальную **специальную способность**, срабатывающую на каждый третий ход

## Особенности реализации
- **Абстрактный класс** `Hero` с наследниками `Knight` и `Wizard`
- **Интерфейс** `IArtifact` для артефактов (например, `HealingPotion`, `PoisonDagger`) — опционально
- **Цветной вывод** и **системные звуки**, адаптированные под ОС (Windows / Linux / macOS)
- Автоматическое **сохранение лога боя** в файл `battle_log.txt` в папку **Документы** пользователя

## Сборка и запуск
Требуется установленный **.NET SDK**.
