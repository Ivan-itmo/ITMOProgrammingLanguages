# Практическое занятие №1: Шифратор Цезаря на языке C

## Цели занятия
- Настроить рабочее окружение для разработки на C.
- Изучить базовую структуру программы на языке C.
- Научиться обрабатывать аргументы командной строки (`argc`, `argv`).
- Реализовать алгоритм шифрования Цезаря с использованием строк и циклов.
- Освоить рефакторинг — вынос логики в отдельные функции.

## Подготовка окружения
- **Windows**: Установите Visual Studio Community с компонентом «Разработка классических приложений на C++».
- **Linux/macOS**: Установите компилятор через `build-essential` (Linux) или `xcode-select --install` (macOS). Рекомендуется использовать Visual Studio Code.

## Теоретическая часть
Директива `#include` — это инструкция для препроцессора, которая подключает заголовочные файлы (например, `stdio.h`). В них содержатся объявления стандартных функций, таких как `printf`, что позволяет компилятору корректно их использовать.

## Задача
Реализовать программу `caesar`, которая принимает два аргумента: строку для шифрования и числовой ключ сдвига. Программа должна шифровать только латинские буквы (верхний и нижний регистр), оставляя остальные символы без изменений.

## Этапы выполнения
1. Написать базовую версию программы полностью в функции `main`.
2. Провести рефакторинг: вынести логику шифрования в отдельную функцию `encrypt`.
3. Убедиться, что программа корректно компилируется и работает.

## Дополнительное задание
Реализовать функцию `decrypt`, которая выполняет обратное преобразование, и протестировать пару «шифрование → дешифрование» для проверки корректности.
