# Практическое занятие №4: Магия LINQ — Декларативная обработка данных в C#

Практическое занятие по работе с коллекциями и языковой интегрированной запросами (LINQ) в .NET.

## Цель
Научиться эффективно обрабатывать и анализировать данные с помощью **LINQ**, используя декларативный стиль вместо императивных циклов.

## Основные темы
- Коллекции в .NET: `List<T>`, массивы, `Dictionary<TKey, TValue>`
- Интерфейс `IEnumerable<T>` — основа для всех LINQ-операций
- **Отложенное выполнение** (lazy evaluation)
- Чтение и парсинг CSV-файлов в строго типизированные объекты
- Ключевые операторы LINQ:
  - `Where` — фильтрация
  - `Select` — проекция и трансформация
  - `OrderBy` / `Take` — сортировка и ограничение
  - `GroupBy` — группировка данных
  - `Any`, `All`, `Average` — агрегация и проверка условий

## Практическая задача
Анализ набора данных о фильмах (`movies.csv`):
- Найти фильмы Тарантино
- Получить топ-5 фильмов после 2010 года
- Найти режиссёра с наибольшим числом фильмов

## Требования
- Установленный **.NET SDK** (6.0 или новее)
- Любой текстовый редактор (VS Code, Rider, и т.д.)
