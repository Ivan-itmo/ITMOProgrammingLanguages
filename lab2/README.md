# Практическое занятие №2: Основы низкоуровневого программирования на C

## Цель
Освоить работу с памятью в языке C: указатели, динамическое выделение памяти и агрегатные типы данных (структуры).

---

## Основные темы

### 1. Указатели и адресная арифметика
- Объявление указателей на базовые типы (`int`, `char`, `float`).
- Операторы `&` (взятие адреса) и `*` (разыменование).
- Итерация по массиву с помощью указателей: `*(arr + i)` ↔ `arr[i]`.
- Разница между `ptr++` и `(*ptr)++`.

### 2. Динамическое управление памятью
- Стек vs куча: автоматическое и ручное управление.
- Функции:  
  - `malloc` / `calloc` — выделение памяти,  
  - `realloc` — изменение размера,  
  - `free` — освобождение.
- Опасности: утечки памяти и висячие указатели.
- Обязательное использование `sizeof` при выделении памяти.

### 3. Агрегатные типы данных
- **Структуры (`struct`)** — объединение связанных данных (пример: `GameUnit` с именем, здоровьем, позицией и типом).
- **Перечисления (`enum`)** — семантические константы (например, `WARRIOR`, `ARCHER`).
- Доступ к полям: `.` для переменных, `->` для указателей.
- Краткое знакомство с `union` и битовыми полями.

---

## Практические задачи
- Работа с адресами переменных и массивов через указатели.
- Создание и расширение динамических массивов.
- Реализация функций для работы с динамической памятью и структурами.
- Форматированный вывод данных о пользовательских типах.
