# Практическое занятие №6: Интеграция Python и C

Демонстрация вызова C-функций из Python с помощью модуля `ctypes` и сравнение производительности (рекурсивный Фибоначчи, обработка структур).

## Сборка C-библиотеки в CLion

1. **Создайте новый проект в CLion**:
   - Тип: **C Library** → **Shared Library**
   - Имя: `fast_lib`

2. **Замените содержимое `fast_lib.c`** на код из задания (`fib_c`, `fib_c_iterative`, `process_point`).
